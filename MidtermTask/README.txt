Project Title: AutoMarket – Car Marketplace Web Application

This project is a web-based car marketplace similar to Kolesa.kz. The purpose of the application is to provide users with a simple and convenient platform to search and add car listings. Users will be able to view detailed information about vehicles.


Features:
Home page with car listings
Car card component (image, title, price, year)
Basic filtering (by brand)

How to run project:
1.You should install src folder from my github repository.
2.Create react app
3.Replace current src folder with my folder.
4.Open cmd and start project.
